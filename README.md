# HOSPITAL MANAGEMENT SYSTEM
 GUI hospital management system made in python.
 This is a class 12 school projrct.

 ## Patinet Side

 A Patient can look for available beds and book a bed according to need and a bill is generated in PDF format.
 Upon booking a unique id is generated which can be used to get paitent details in future.

 ## Staff Side

 In staff side, one can enter the id of a patient and get the patient details,update the bed,
 ,print details of the patient and release the patient.

 You can also get the bed details table as well as patient details table.

You also have the option to book a bed for the patient in the staff side.
